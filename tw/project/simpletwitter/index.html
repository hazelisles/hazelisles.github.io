簡單版推特

Posted on Tuesday, 2021-08-03  •  1 minutes  • 401 words  • Other languages:  EN

Source Code at Github

第一次經歷多人協作，藉由這次經驗，更了解怎麼處理衝突的程式碼、怎麼設分支進行開發再 pull request 進行合併、review 程式碼給予回饋等等。

我主要的分配任務包括

  1. 註冊、登入、登出的頁面及功能
  2. 後臺管理的所有頁面及功能，查看使用者、管理員狀態，刪除特定 tweet 等等
  3. 協助優化所有頁面呈現一致風格及增加支援不同大小螢幕的呈現(RWD)
  4. 彈跳視窗的切版
  5. 個人資料設定與更新的功能

專案使用的主要工具包括

前端：HTML5, CSS3, Bootstrap5, JavaScript, handlebars

後端：Express.js, sequelize, MySQL, bcryptjs, session, passport

其他：socket.io, Heroku
